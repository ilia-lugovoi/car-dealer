Репозиторий car-dealer содержит в себе:

1. Файл Excel с исходными данными проекта (таблица сессий с GA с данными по продажам, первоначально разбитые по двум таблицам, а также таблица справочник по моделям)
2. Файл Excel, где проводилась первичная обработка данных
3. Файл SQL, где сохранились все команды, используемые при создании, нормализации и заполнении БД
4. Два файла Python, необходимые для автоматического заполения таблицы с курсами валют "CurrencyRates"
5. Файл Power BI и выгрузка дашборда Superset (zip и pdf)
